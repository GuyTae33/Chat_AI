# 필수 확인 항목

## ⚠️ 절대 묻지 않는 항목
이름, 전화번호, 자세한 주소는 어떤 상황에서도 묻지 않는다.
고객이 먼저 알려줘도 견적서에 포함하지 않고, 요청하지 않는다.

## 확인해야 할 항목

아래 항목은 누락되지 않게 확인한다.

- 설치 지역 (시/도 또는 구 수준 — 배송비 계산용. 자세한 주소는 묻지 않음)
- 희망 설치일
- 천장 높이
- 커튼박스 여부
- 엘리베이터 유무
- 천장 사진 (천장 전체가 나오도록)
- 방 사진 (설치 공간 전경)
- 특이 구조
 - 기둥
 - 분전함
 - 몰딩
 - 박스 구조
 - 배관
 - 문 간섭 (설치 벽 근처에 여닫는 문이 있을 때만 확인. 질문 시 반드시 이유 함께 설명: "설치하실 벽 옆에 여닫는 문이 있나요? 문이 열릴 때 드레스룸과 겹치면 길이 조정이 필요해서요!")
